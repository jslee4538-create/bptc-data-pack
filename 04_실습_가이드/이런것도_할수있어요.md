### 예시 1. 서면역 시간대별 승차 패턴

**질문**: 서면역에서 사람이 가장 많이 타는 시간은 언제인가.

**결과**
```
6시    289명
7시    526명
8시    925명
9시    587명
```
8시가 6시의 3배. 출근 시간 한 점에 집중.

**SQL**
```sql
SELECT hour, ROUND(AVG(passengers), 0) as 평균_승차
FROM daily_passengers
WHERE station_name = '서면'
  AND direction = 'board'
GROUP BY hour ORDER BY hour;
```

### 예시 2. 노선별 누적 승차 비교

**질문**: 1호선이 부산 대중교통에서 차지하는 비중.

**결과**
```
1호선   5,141만 명  (전체의 49%)
2호선   4,008만 명  (38%)
3호선   1,165만 명
4호선     336만 명
```
1호선 혼자 부산 대중교통의 절반.

**SQL**
```sql
SELECT line_no,
       ROUND(SUM(passengers) / 1000000.0, 1) as 누적_백만명
FROM daily_passengers
WHERE direction = 'board'
GROUP BY line_no
ORDER BY 누적_백만명 DESC;
```

### 예시 3. BIFF 기간 해운대역 변화

**질문**: 부산국제영화제 기간 동안 해운대역 승하차가 평소 대비 얼마나 늘었나.

**결과**
```
평소 평균      25,000명/일
9/17 개막일    38,900   (+55%)
9/20 주말 피크 42,300   (+70%)
9/26 폐막일    36,700   (+47%)
```
BIFF 기간 평균 +52% 증가.

**SQL**
```sql
SELECT date, SUM(passengers) as 일_합계
FROM daily_passengers
WHERE station_name = '해운대'
  AND direction = 'board'
  AND date BETWEEN '2025-09-15' AND '2025-09-30'
GROUP BY date ORDER BY date;
```

### 예시 4. 자치구 5년 시세 추이

**질문**: 부산 아파트 평균 거래금액이 5년 동안 어떻게 변했나.

**결과 (부산 전체 평균)**
```
2021년   3.1억
2023년   4.0억   (+29%)
2025년   4.5억   (+45%)
2026년   4.3억   (-4%, 조정)
```
5년 동안 평균 1.4억 상승.

**SQL**
```sql
SELECT SUBSTR(deal_date, 1, 4) as 연도,
       COUNT(*) as 거래수,
       ROUND(AVG(dealAmount) / 10000.0, 2) as 평균_억원
FROM apt_trade
WHERE gu_name = '해운대구'
GROUP BY 연도 ORDER BY 연도;
```

### 예시 5. 단지별 5년 가격 변동

**질문**: 해운대 엘시티 가격이 5년 동안 어떻게 변했나.

**결과**
```
2021년   43.5억   (1건)
2022년   38.6억   (5건)
2023년   34.1억   (19건)
2025년   31.2억   (51건)
```
5년 동안 -28% 하락. 거래량은 증가.

**SQL**
```sql
SELECT SUBSTR(deal_date, 1, 4) as 연도,
       COUNT(*) as 거래수,
       ROUND(AVG(dealAmount) / 10000.0, 1) as 평균_억원,
       ROUND(MAX(dealAmount) / 10000.0, 1) as 최고_억원
FROM apt_trade
WHERE aptNm LIKE '%엘시티%'
GROUP BY 연도 ORDER BY 연도;
```

### 예시 6. 학원가 데이터로 본 학군

**질문**: 입시강좌 수로 보면 부산 학군 1위 자치구는 어디인가.

**결과**
```
자치구       학원수   입시강좌   평균교습비
─────────────────────────────────────
동래구        697     6,228     267,473원
해운대구      583     4,948     268,401
부산진구      876     3,816     290,108
사하구        682     4,655     223,840
```
학원 수는 부산진구가 1위, 입시강좌 수는 동래구가 1위.

**SQL**
```sql
SELECT gu_name,
       COUNT(DISTINCT academy_name) as 학원수,
       SUM(ipsi_count) as 입시강좌,
       ROUND(AVG(avg_tuition), 0) as 평균교습비
FROM academies
GROUP BY gu_name
ORDER BY 입시강좌 DESC LIMIT 8;
```

### 예시 7. 코스피 폭등일 외국인 매매 분석

**질문**: 2026-05-06 코스피 8000 돌파일에 외국인이 어떤 종목을 매수했나.

**결과**
```
종목          외국인 순매수    당일 등락
─────────────────────────────────────
삼성전자       30,849억        +14.41%
SK하이닉스      1,928억        +10.64%
현대차          1,553억         +2.04%
LG전자          1,484억         +8.17%
미래에셋증권      828억        +19.20%
```
삼성전자 외국인 순매수 3조, 당일 +14% 상승.

**SQL (3-테이블 JOIN)**
```sql
SELECT s.name,
       ROUND(i."foreign" / 100000000.0, 1) as 외국인순매수_억원,
       ROUND(d.change_pct, 2) as 등락률
FROM investor_flow i
JOIN stocks_master s ON i.code = s.code
JOIN daily_price d   ON i.code = d.code AND i.date = d.date
WHERE i.date = '2026-05-06' AND i."foreign" > 0
ORDER BY i."foreign" DESC LIMIT 5;
```

### 예시 8. 종목별 외국인 보유율 추이

**질문**: 삼성전자 외국인 보유율이 5년 동안 어떻게 변했나.

**결과 (매년 5월말 기준)**
```
2021-05   52.1%
2023-05   51.4%
2025-05   50.8%
2026-05   49.4%
```
5년 동안 외국인 보유율 -2.7%p.

**SQL**
```sql
SELECT m.date, ROUND(m.foreign_rate, 2) as 외국인보유율
FROM daily_marcap m
JOIN stocks_master s ON m.code = s.code
WHERE s.name = '삼성전자'
  AND m.date LIKE '____-05-21'
ORDER BY m.date;
```

### 예시 9. 코스피 vs 코스닥 5년 수익률

**질문**: 코스피 지수와 코스닥 지수 5년 수익률 차이.

**결과**
```
코스피  (1001)   2021-01 → 2026-05    +95%
코스닥  (2001)   2021-01 → 2026-05    +28%
```
같은 기간 코스피가 코스닥 대비 3배 이상 상승.

**SQL**
```sql
WITH s AS (
    SELECT index_code, MIN(date) as min_d
    FROM index_ohlcv WHERE index_code IN ('1001','2001')
    GROUP BY index_code),
e AS (
    SELECT index_code, MAX(date) as max_d
    FROM index_ohlcv WHERE index_code IN ('1001','2001')
    GROUP BY index_code)
SELECT i.index_name,
       ROUND(p1.close, 0) as 시작가,
       ROUND(p2.close, 0) as 종가,
       ROUND((p2.close - p1.close) * 100.0 / p1.close, 1) as 수익률_pct
FROM s JOIN e ON s.index_code = e.index_code
JOIN indexes i ON s.index_code = i.index_code
JOIN index_ohlcv p1 ON s.index_code = p1.index_code AND s.min_d = p1.date
JOIN index_ohlcv p2 ON s.index_code = p2.index_code AND e.max_d = p2.date;
```

## 활용 안내

**언제 보여주나**
- 강의 중간 "예시 잠깐" 5분 끼워넣기
- 1:1 코칭 시작 시 "9개 중 어디 끌리는지" 질문지
- 강의 후 카톡 — "집에서 본인 데이터로 변형해보기" 숙제

**수강생 변형 방법**
- SQL의 자치구·종목·역 이름을 본인 관심사로 교체
- 컬럼명·테이블명은 02_데이터베이스/schemas/ 참조
- 막히면 AI에게 SQL과 함께 "이걸 본인 종목으로 바꿔줘" 요청

**5단계 매핑** (헌법 §2)
- 1단계 질문하기: 9개 예시 모두 질문에서 시작
- 3단계 만들기: SQL 작성 후 차트화
- 5단계 개선하기: 본인 데이터로 변형
