## 강의 시간 배분 (3시간 1회차 — 3-DB 버전)

| 구간 | 시간 | 내용 |
|------|------|------|
| 오프닝 + 킬러 데모 | 20분 (11%) | "수영구가 해운대보다 평당 600만원 비싸다" + "외국인 3조 → 삼성 +14%" |
| 이론 (SQL 기본) | 25분 (14%) | DB·테이블·SELECT/WHERE/GROUP BY/JOIN — 비유 중심 |
| 핵심 실습 1: 부산 도시철도 | 45분 (25%) | 출퇴근 역 패턴 → Folium HTML |

## 실습 1: "내가 자주 타는 역의 진짜 패턴" (70분)

### 1️⃣ 질문하기 (10분)
**수강생 질문**: "내가 매일 타는 역, 진짜 출근시간이 제일 붐비나? 다른 시간대는?"

수강생이 AI(Claude/ChatGPT)에게:
- "bptc_busan.db에 daily_passengers 테이블이 있고 컬럼은 [station_name, hour, direction, passengers]야. 서면역 시간대별 평균 승차 인원 SQL 짜줘"
- AI 응답을 그대로 받기

### 2️⃣ 기획하기 (10분)
**대시보드 4분할 설계** (수강생이 결정):
```
┌──────────────────┬──────────────────┐
│  ① 시간대별 라인 │  ② 요일별 막대   │
│  (출퇴근 피크?)  │  (평일 vs 주말)  │
├──────────────────┼──────────────────┤
│  ③ TOP10 역 막대 │  ④ 노선별 점유   │
│  (어디가 1위?)   │  (1호선이 절반?) │
└──────────────────┴──────────────────┘
```

### 3️⃣ 만들기 (35분) ★ 핵심 단계
**산출물**: Folium HTML 파일 1개 (본인 자주 가는 역 중심)

SQL → pandas → matplotlib/Folium 흐름 (강사 시연 10분 + 수강생 따라하기 25분):
```python
import sqlite3, pandas as pd
conn = sqlite3.connect('bptc_busan.db')
df = pd.read_sql("""
    SELECT hour, AVG(passengers) as avg_pass
    FROM daily_passengers
    WHERE station_name = '서면' AND direction = 'board'
    GROUP BY hour ORDER BY hour
""", conn)
df.plot(x='hour', y='avg_pass', kind='line')
```

### 4️⃣ 검토하기 (10분)
AI에게 차트 스크린샷 첨부 + 질문:
- "이 차트로 보면 5~6시에 갑자기 사람이 늘어. 통계적으로 의미 있는 차이일까?"
- "내 역만 보면 8시 피크인데, 다른 역들과 비교하면 평범한 패턴일까?"

수강생이 AI 답변을 비판적으로 평가 (그대로 믿지 말 것).

### 5️⃣ 개선하기 (5분)
"다음에 무엇을 더 분석할까?" — 다른 데이터 결합 아이디어:
- 부산 자치구 인구 (`gu_codes` + `population_gu`)
- 날씨 (기상청 데이터 추가)
- BIFF 기간 해운대역 효과 (`WHERE date BETWEEN '2025-09-17' AND '2025-09-26'`)

## 실습 2: "내 동네 아파트 5년 동안 얼마나 올랐어?" (50분) ★ 부산 부동산

### 1️⃣ 질문하기 (8분)
**수강생 질문**: "내가 사는 (또는 사고 싶은) 자치구 아파트 평균 가격, 5년 동안 어떻게 변했어?"

본인 거주 자치구 선택. 부산진구·해운대구·수영구 등.

### 2️⃣ 기획하기 (7분)
**대시보드 4분할 설계**:
```
┌──────────────────┬──────────────────┐
│  ① 연도별 평균    │  ② 단지 TOP 10  │
│  거래금액 라인    │  (가장 비싼)     │
├──────────────────┼──────────────────┤
│  ③ 거래량 막대   │  ④ 평형별 분포  │
│  (월별)          │  (산점도)        │
└──────────────────┴──────────────────┘
```

### 3️⃣ 만들기 (25분)
```python
import sqlite3, pandas as pd, plotly.express as px
conn = sqlite3.connect('bptc_realestate.db')
df = pd.read_sql("""
    SELECT
      SUBSTR(deal_date, 1, 7) as 연월,
      ROUND(AVG(dealAmount)/10000.0, 2) as 평균_억원,
      COUNT(*) as 거래수
    FROM apt_trade
    WHERE gu_name = '해운대구'    -- 본인 자치구로 변경
    GROUP BY 연월
    ORDER BY 연월
""", conn)
fig = px.line(df, x='연월', y='평균_억원', title='해운대구 아파트 평균 거래금액 5년 추이')
fig.write_html('해운대_5년추이.html')
```

### 4️⃣ 검토하기 (7분)
AI에게:
- "엘시티가 5년 동안 -28% 떨어졌어. 다른 단지도 비슷해?"
- "신축이 진짜 비싼가? 90년대 구축과 비교해줘"

### 5️⃣ 개선하기 (3분)
- 다른 자치구 추가 비교
- 전세가율 계산 (매매가 vs 전세가 비교)
- 지하철역 인접 단지만 필터 (bptc_busan.db와 결합)

## 실습 2-2 (심화): "동래구는 진짜 학군 1위인가?" (20분, 시간 여유 시)

> 단순 거리 ≠ 학군. **입시강좌 수·교습비·학원 밀도**로 정량 평가.

### 1️⃣ 질문하기 (3분)
**수강생 질문**: "사람들은 '동래구가 부산 학군 1위'라 하는데 데이터로도 그런가?"

### 2️⃣ 기획하기 (2분)
4가지 지표 한 화면 비교:
- 학원 수 / **입시 강좌 수** (진짜 학군 지표) / 평균 교습비 / 평균 거래가

### 3️⃣ 만들기 (10분)
```sql
SELECT
    a.gu_name,
    COUNT(DISTINCT a.academy_name) as 학원수,
    SUM(a.ipsi_count) as 입시강좌수,
    ROUND(AVG(a.avg_tuition), 0) as 평균교습비_원,
    (SELECT ROUND(AVG(dealAmount)/10000.0, 1)
       FROM apt_trade WHERE gu_name = a.gu_name AND deal_date >= '2025-01-01') as 평균거래_억원
FROM academies a
GROUP BY a.gu_name
ORDER BY 입시강좌수 DESC LIMIT 8;
```

### 4️⃣ 검토하기 (3분)
검증된 결과:
- **동래구 입시강좌 6,228개로 1위** — 통념과 일치
- **부산진구는 학원 수 1위(876)지만 교습비도 1위(29만원)** — "비싼 학원가"
- **수영구는 평균거래 1위인데 학원 약함** → 학군 프리미엄이 아님
- 본인 동네 학원·집값 데이터로 판단

### 5️⃣ 개선하기 (2분)
- 단지 1km 반경 학원 수 계산 (위경도 필요)
- 학원 분야별(국어/수학/영어) 자치구 차이

## 실습 3 (선택): "내 종목, 외국인이 사고 있나?" (30분)

### 1️⃣ 질문하기 (8분)
**수강생 질문**: "내가 산 (또는 살까 말까 하는) 종목 외국인이 늘리고 있어, 줄이고 있어? 5년 동안 어떻게 변했지?"

본인 보유종목 or 관심종목 1개 선택 (없으면 삼성전자/SK하이닉스/카카오).

### 2️⃣ 기획하기 (7분)
**차트 1개로 답하기**:
```
삼성전자 ─── 종가 (좌측 축, 만원)
          ┌─ 외국인 보유율 (우측 축, %)
2021 ──────────────────────── 2026
```
- 종가 라인 (위)
- 외국인 보유율 라인 (아래) — 이중축
- 한 화면에서 "주가 vs 외국인" 보이게

### 3️⃣ 만들기 (25분)
```python
import sqlite3, pandas as pd, plotly.graph_objects as go
conn = sqlite3.connect('bptc_stocks.db')
df = pd.read_sql("""
    SELECT d.date, d.close, m.foreign_rate
    FROM daily_price d
    JOIN daily_marcap m ON d.code = m.code AND d.date = m.date
    JOIN stocks_master s ON d.code = s.code
    WHERE s.name = '삼성전자'
    ORDER BY d.date
""", conn)
# Plotly 이중축 차트 → HTML 저장
```

### 4️⃣ 검토하기 (7분)
AI에게:
- "외국인 보유율이 1년 동안 2%p 떨어졌어. 의미 있는 변화야?"
- "주가는 올랐는데 외국인은 팔았어. 이게 모순 아니야?"

## 5단계 매핑 체크리스트 (강사 사전 점검)

```
□ 모든 실습에 1~5단계 라벨 명시됨
□ 1단계(질문하기)에서 수강생이 본인 데이터/관심사로 시작
□ 3단계(만들기)에서 수강생 산출물 만들어짐
□ 4단계(검토하기)에서 AI 답변을 비판적으로 평가
□ 5단계(개선하기)에서 다음 행동 1개 이상 제시
□ 이론 30% 이하 유지
□ 모든 인사이트에 데이터 출처 표시
```
