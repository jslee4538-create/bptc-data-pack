# bptc_realestate.db — 부산 부동산 데이터베이스

> 국토교통부 실거래가 공개 API (공공데이터포털)
> 부산 16자치구 × 5년 (2021-01 ~ 2026-05) 데이터
> 공공누리 1유형 · 상업적 이용 가능

## 데이터 한계 (정직한 안내)

- **연립·다세대·단독 거래**: API 활용신청 별도 필요 (아파트만 받음)
- **공공누리 1유형** — 상업적 사용 가능, 출처 표시 권장
- 거래일 외 단지 좌표는 별도 결합 필요 (도로명 → 좌표 변환 또는 카카오 API)
- 학교·의료·인구 테이블은 별도 CSV 다운로드 후 채워야 함

## 다른 DB와 결합

```sql
-- 부산 부동산 + 부산 지하철 결합 (자치구 코드 공통)
ATTACH DATABASE 'bptc_busan.db' AS busan;
SELECT ar.gu_name, AVG(ar.dealAmount) as avg_price,
       COUNT(DISTINCT b.station_name) as 지하철역_수
FROM apt_trade ar
JOIN busan.stations b ON ar.gu_name = ...
GROUP BY ar.gu_name;
```
