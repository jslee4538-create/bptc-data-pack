## 5분 시작 가이드

### 1단계 · 압축 풀기

```bash
cd 02_데이터베이스
unzip 압축/bptc_all.zip
```

`bptc_busan.db`, `bptc_stocks.db`, `bptc_realestate.db` 3개 파일이 같은 폴더에 생깁니다.

### 2단계 · Claude Code 열기

이 폴더에서 Claude Code를 켭니다.

### 3단계 · DB 구조 읽히고 질문

Claude Code 첫 메시지에 `03_AI활용/CLAUDE.md`를 읽힙니다. 이 파일에는 데이터베이스에
어떤 표·컬럼이 있는지 정리돼 있어서, **AI가 DB 구조를 파악하고** 정확히 조회합니다.

```
03_AI활용/CLAUDE.md를 읽고 이 데이터베이스 구조를 파악해줘.
그다음 bptc_busan.db에서 서면역 시간대별 평균 승차 인원을 표로 보여줘.
```

Claude Code가 구조를 파악한 뒤 알아서 데이터를 조회해 결과를 정리합니다.
SQL을 짜거나 들여다볼 필요 없이, 질문만 던지면 됩니다.

> 무엇을 물어볼지 막막하면 —
> "이 데이터베이스로 뭘 물어볼 수 있는지 예시 알려줘."로 시작해도 됩니다.

## AI 활용 흐름

SQL을 직접 작성하지 않습니다. 질문을 던지고 결과를 받습니다.

```
1. 03_AI활용/CLAUDE.md를 읽혀 DB 구조 파악
2. 자연어로 질문 ("해운대구 5년 시세 추이")
3. Claude Code가 알아서 데이터 조회 (쿼리는 AI가 작성)
4. 결과 → "차트 HTML로 만들어줘" 요청
```

자세한 프롬프트 패턴은 `03_AI활용/프롬프트_템플릿.md`.

## 데이터 한계

### 부산 도시철도
- 1~4호선만 (동해선·부산김해경전철 제외)
- 시간 단위 1시간
- stations 테이블의 위경도 비어있음 (지도 시각화 시 별도 수집 필요)

### 한국 주식
- 공매도 데이터 없음 (KRX 정책상 일반 IP 차단)
- 분 단위·실시간 없음 (일별만)
- 펀더멘털은 월말 기준

### 부산 부동산
- 아파트 매매·전월세만 (연립·단독은 API 활용신청 별도 필요)
- 5년치 (2021-01 ~ 2026-05)
- 학원 데이터는 강좌 단위 (한 학원이 여러 강좌 보유 가능)
